## Certificates with chocolatey and mkcert

- Run PowerShell as administrator, execute:
- > Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::
  SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object
  System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
- > choco install mkcert
- > mkcert -key-file cert.key -cert-file cert.crt localhost
